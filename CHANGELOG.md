# Changelog

All notable changes to SoilGuard Portal are documented here.

## [1.2.0] - 2026-06-08

### Added
- Initial repository structure and portal scaffold
- NZ regulatory compliance framework (NEMS-SQ 2022/2025, NES-F, Freshwater Farm Plans)
- Multi-modal sensor pipeline (pH, moisture, temperature via MQTT)
- Gemma 4 multimodal inference via Ollama (gemma4:e4b)
- Camera-based ground cover and erosion risk detection
- LangGraph orchestration layer
- Compliance export module (JSON/CSV - Farm Plan and SoE ready)
- Systemd service unit (soilguard.service)
- HARDWARE_SETUP.md with NZ-specific probe and calibration guidance
- COMPLIANCE.md with NEMS-SQ, NES-F, and Freshwater Farm Plan regulatory mapping
